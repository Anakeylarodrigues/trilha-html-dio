O que são Clients

Voltar ao índice

"Client" é um termo empregado em computação e representa uma entidade que consome os serviços de uma outra entidade servidora, em geral através do uso de uma rede de computadores numa arquitetura cliente-servidor.

*Para saber mais acesse Wikipedia

O que são Servers

Voltar ao índice

Um servidor é um computador equipado com um ou mais processadores, bancos de memória, portas de comunicação, softwares e, ocasionalmente, algum sistema para armazenamento de dados, como hard disks internos ou memórias SSD. O termo servidor também define um recurso dentro de um sistema computacional maior, capaz de processar aplicações, prestar serviços e armazenar dados. Esses sistemas podem ser físicos ou virtuais, estarem instalados local ou remotamente. Normalmente, são escaláveis e possuem alto poder de processamento.

*Para saber mais acesse Controle.net

O que é HTML

Voltar ao índice

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo.
HTML não é uma linguagem de programação; é uma linguagem de marcação, usada para definir a estrutura do seu conteúdo. HTML consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira.

*Para saber mais acesse Developer.mozilla.org

Diferença entre listas ordenadas e não ordenadas

Voltar ao índice

Muito do conteúdo da web é de listas e o HTML tem elementos especiais para elas. Listas de marcação sempre consistem em pelo menos 2 elementos. Os tipos mais comuns de lista são ordenadas e não ordenadas:

  1. Listas não ordenadas são para listas onde a ordem dos itens não importa, como uma lista de compras.
  2. Listas Ordenadas são para listas onde a ordem dos itens importa, como uma receita.

*Para saber mais acesse Developer.mozilla.org
